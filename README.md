# Innsikt — Frontend

Data visualization platform for Norwegian police log data from the public [Politiloggen API](https://api.politiet.no/politiloggen/v1).

## Stack

- **Next.js 16** — App Router, TypeScript
- **Tailwind v4**
- **ECharts** — data visualizations
- **Leaflet** — interactive incident map
- **Radix UI** — accessible primitives
